## 2. 字体

策略：**西文用 Web Font，中文用系统字体栈**（中文字体体积大，不适合 Web 加载）。

### 2.1 正文字体（Sans-serif）

| 平台 | 西文 | 中文 | 后备 |
|------|------|------|------|
| **macOS / iOS** | Inter (Web Font) | 苹方 `PingFang SC` | 系统 `system-ui` |
| **Windows** | Inter (Web Font) | 微软雅黑 `Microsoft YaHei` | 思源黑体 `Noto Sans SC` |
| **Android** | Inter (Web Font) | 思源黑体 `Noto Sans SC` | `sans-serif` |
| **fallback** | — | — | `sans-serif` |

CSS 栈：
```
font-family: 'Inter', -apple-system, 'PingFang SC', 'Microsoft YaHei', 'Noto Sans SC', sans-serif;
```

### 2.2 代码字体（Mono）

| 字体 | 平台 |
|------|------|
| JetBrains Mono (Web Font) | 所有平台优先加载 |
| Fira Code | 后备 |
| `Menlo`, `Consolas` | 系统字体后备 |
| `monospace` | 最终后备 |

```
font-family: 'JetBrains Mono', 'Fira Code', Menlo, Consolas, monospace;
```

### 2.3 字号阶梯（建议）

| 层级 | 字号 | 行高 | 用途 |
|------|------|------|------|
| H1 | 48px / 3rem | 1.15 | 页面主标题 |
| H2 | 36px / 2.25rem | 1.2 | 区块标题 |
| H3 | 28px / 1.75rem | 1.25 | 子标题 |
| Body | 16px / 1rem | 1.6 | 正文 |
| Small | 14px / 0.875rem | 1.5 | 辅助信息 |
| Caption | 12px / 0.75rem | 1.4 | 注释/标签 |

> 响应式缩放：移动端所有字号 × 0.875

---

> 选定后通过 `@fontsource` 安装 Inter 和 JetBrains Mono，配置到 Tailwind 中。

## 3. 间距与布局

### 3.1 基础间距单位（已确认：均衡）

采用 **8px 栅格系统**（Tailwind 默认 `0.25rem = 4px`，偶数倍取 8px 的倍数）。

| Tailwind 类 | 值 | 用途 |
|-------------|-----|------|
| `space-2` | 8px | 内元素间距（图标与文字、列表项） |
| `space-3` | 12px | 紧凑间距 |
| `space-4` | 16px | 卡片内边距、段落间距 |
| `space-5` | 20px | 组件间距 |
| `space-6` | 24px | 中等间距 |
| `space-8` | 32px | 区块间距 |
| `space-12` | 48px | 大区块间距 |
| `space-16` | 64px | 页面级留白 |

### 3.2 内容最大宽度

| 断点 | 值 | 用途 |
|------|-----|------|
| `max-w-prose` | 65ch | 正文阅读区（最佳可读性） |
| `max-w-2xl` | ~672px | 卡片/表单 |
| `max-w-4xl` | ~896px | 页面容器（窄屏） |
| `max-w-6xl` | ~1152px | 页面容器（宽屏） |

### 3.3 响应式断点

| 断点 | 值 | 设备 |
|------|-----|------|
| `sm` | 640px | 手机横屏 |
| `md` | 768px | 平板竖屏 |
| `lg` | 1024px | 平板横屏 / 小笔记本 |
| `xl` | 1280px | 桌面 |
| `2xl` | 1536px | 大屏 |

### 3.4 页面留白策略

- PC：左右对称留白，内容居中
- 移动端：左右各 16px padding（`sm:px-4`）
- 上下间距：区块之间 `py-16`（64px），移动端 `py-10`

## 4. 组件风格（已确认）

### 4.1 边框半径

| 用途 | 值 | 类 |
|------|-----|-----|
| 按钮 | `12px` | `rounded-xl` |
| 输入框/标签 | `8px` | `rounded-lg` |
| 卡片 | `12px` | `rounded-xl` |
| 模态框/弹窗 | `12px` | `rounded-xl` |

> 按钮和卡片用大圆角，输入框用微圆角，形成层级区分。

### 4.2 阴影与边框

原则：**边框为主、阴影为辅**，既有质感又不飘。

| 层级 | 方案 |
|------|------|
| 卡片 | 1px 浅色边框 + `shadow-sm`（`0 1px 2px rgba(0,0,0,0.05)`） |
| 悬浮元素 | hover 时边框颜色加深 + 微上移 |
| 弹窗/下拉 | 1px 边框 + `shadow-md` |

### 4.3 按钮

| 状态 | 样式 |
|------|------|
| 主要按钮 | 强调色背景 + 白字 + 大圆角 |
| 次要按钮 | 透明背景 + 强调色文字 + 细边框 + 大圆角 |
| 禁用 | 50% 透明度 + `cursor-not-allowed` |

### 4.4 卡片

- 1px 浅色边框 + 微阴影 `shadow-sm`
- 背景与页面背景同色或深一档
- hover 时边框加深 + 微上移 `translateY(-2px)`

## 5. 动效（已确认：柔和轻快之间）

原则：**克制、流畅、不炫技**。全部采用 200ms，介于轻快（150ms）和柔和（300ms）之间。

| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮 hover | 200ms | `ease-out` |
| 卡片 hover | 200ms | `ease-out` + 上移 2px |
| 页面切换 | 200ms | `ease-in-out` |
| 展开/折叠 | 200ms | `ease-out` |

> 遵循系统 `prefers-reduced-motion`：用户关闭动效时全部禁用。
