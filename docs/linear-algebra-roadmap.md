# 线性代数学习路线图

> 线性代数是机器学习、计算机图形学、机器人学、物理学等领域的数学基础。掌握它不是"知道怎么算"，而是"真正理解每个概念在做什么"。

## 阶段一：向量与空间直觉

建立对"向量"和"空间"的直觉理解，不要一上来就陷入矩阵运算。

### 1. 向量的本质

- [ ] 向量是什么：不是"有方向的箭头"，而是**空间中的一个点**，或**一组有序的数字**
- [ ] 向量的两种视角：
  - 几何视角：箭头、方向、长度
  - 数据视角：一组坐标、一个列表
- [ ] 向量的基本运算：加法、数乘
- [ ] 线性组合（Linear Combination）：$c_1 \vec{v}_1 + c_2 \vec{v}_2 + \cdots + c_n \vec{v}_n$
- [ ] 张成空间（Span）：一组向量的所有线性组合构成的集合
- [ ] 线性相关 vs 线性无关：能不能用更少的向量表示同样的东西

### 2. 点积与投影

- [ ] 点积（Dot Product）的定义与几何意义
- [ ] 点积的两种理解：
  - 代数：对应分量相乘再相加
  - 几何：$\vec{a} \cdot \vec{b} = |\vec{a}| |\vec{b}| \cos\theta$
- [ ] 正交（垂直）的判定：点积为零
- [ ] 投影（Projection）：把一个向量"压"到另一个向量上
- [ ] 投影的应用：最小二乘、相似度计算

### 3. 向量空间（Vector Space）

- [ ] 向量空间的定义与公理（封闭性、结合律、交换律等）
- [ ] 子空间（Subspace）：大空间里的小空间
- [ ] 基（Basis）：一组能张成整个空间的线性无关向量
- [ ] 标准基：$\hat{i}, \hat{j}, \hat{k}$
- [ ] 维度（Dimension）：基中向量的个数
- [ ] 坐标变换：同一向量在不同基下的坐标不同

## 阶段二：矩阵——空间变换的语言

矩阵不是"数字表格"，而是**对空间的变换操作**。

### 4. 矩阵与线性变换

- [ ] 矩阵的本质：一种**线性变换**的编码方式
- [ ] 矩阵 × 向量 = 把向量从一个位置变换到另一个位置
- [ ] 矩阵的列 = 基向量变换后的位置（**核心直觉**）
- [ ] 常见的线性变换：
  - 旋转
  - 缩放
  - 剪切（Shear）
  - 反射
  - 投影
- [ ] 矩阵乘法的意义：变换的**复合**（先做 A，再做 B）
- [ ] 为什么矩阵乘法不满足交换律：变换的顺序很重要

### 5. 行列式（Determinant）

- [ ] 行列式的几何意义：变换后面积/体积的**缩放因子**
- [ ] 行列式为零 = 空间被"压扁"了（降维）
- [ ] 行列式为负 = 空间发生了翻转
- [ ] 行列式的性质与计算
- [ ] 行列式与可逆性的关系

### 6. 逆矩阵与列空间

- [ ] 逆矩阵 $A^{-1}$：变换的"撤销"操作
- [ ] 什么样的矩阵可逆？（满秩、行列式非零）
- [ ] 列空间（Column Space）：矩阵所有可能输出构成的空间
- [ ] 零空间 / 核（Null Space / Kernel）：被变换到原点的所有向量
- [ ] 秩（Rank）：列空间的维度
- [ ] 方程组 $A\vec{x} = \vec{b}$ 的解的几何理解：
  - 有唯一解：$\vec{b}$ 在列空间中，且零空间只有原点
  - 无解：$\vec{b}$ 不在列空间中
  - 无穷多解：$\vec{b}$ 在列空间中，且零空间非平凡
- [ ] 高斯消元法与行阶梯形

## 阶段三：深入矩阵的结构

### 7. 特征值与特征向量（Eigenvalues & Eigenvectors）

- [ ] 定义：$A\vec{v} = \lambda \vec{v}$ —— 变换后**方向不变**的向量
- [ ] 特征值 $\lambda$：变换后伸缩了多少倍
- [ ] 特征空间的几何直觉
- [ ] 特征多项式与特征方程
- [ ] 对角化（Diagonalization）：$A = PDP^{-1}$
- [ ] 什么时候可以对角化？
- [ ] 应用：矩阵的幂、微分方程、主成分分析（PCA）

### 8. 正交性与正交基

- [ ] 正交向量与正交矩阵
- [ ] 格拉姆-施密特正交化（Gram-Schmidt）
- [ ] 正交矩阵的性质：$Q^T Q = I$，$Q^{-1} = Q^T$
- [ ] 标准正交基（Orthonormal Basis）
- [ ] 正交投影到子空间
- [ ] 最小二乘问题的正交解释

### 9. 对称矩阵与二次型

- [ ] 对称矩阵的性质：实特征值、正交特征向量
- [ ] 谱定理（Spectral Theorem）：对称矩阵一定可以对角化
- [ ] 正定矩阵、半正定矩阵
- [ ] 二次型：$\vec{x}^T A \vec{x}$
- [ ] 主轴定理

## 阶段四：矩阵分解（核心武器）

### 10. LU 分解

- [ ] $A = LU$：高斯消元的矩阵表达
- [ ] 用途：高效求解线性方程组

### 11. QR 分解

- [ ] $A = QR$：列向量的正交化
- [ ] 与格拉姆-施密特的关系
- [ ] 用途：最小二乘、特征值计算

### 12. 奇异值分解（SVD）—— 最重要的分解

- [ ] $A = U \Sigma V^T$ 的含义
- [ ] 奇异值：矩阵"真正重要"的缩放因子
- [ ] SVD 的几何解释：旋转 → 缩放 → 旋转
- [ ] 低秩近似：用少量奇异值近似原矩阵
- [ ] 应用：
  - 数据降维（与 PCA 的关系）
  - 图像压缩
  - 推荐系统
  - 伪逆（Moore-Penrose Pseudoinverse）

### 13. 特征分解 vs SVD

- [ ] 什么时候用特征分解，什么时候用 SVD
- [ ] 对称矩阵的 SVD 与特征分解的关系

### 学习方法

1. **先直觉后计算**：先看动画理解概念在做什么，再去学怎么算
2. **每个概念问三个问题**：
   - 它是什么？（定义）
   - 它在几何上意味着什么？（直觉）
   - 它能用来做什么？（应用）
3. **手算 + 代码验证**：用 Python (NumPy) 验证你的计算结果
4. **不要跳过"为什么"**：不要只会算行列式，要理解行列式为什么是那个公式
