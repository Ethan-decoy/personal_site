# 函数图像测试

## 三角函数

### cos(x)

```plot
Math.cos(x)
```

### sin(x)

```plot
Math.sin(x)
```

### tan(x)

```plot
Math.tan(x)
```

## 多项式

### x²

```plot
x*x
```

### x³ - 3x

```plot
x*x*x - 3*x
```

## 指数与对数

### e^x

```plot
Math.exp(x)
```

### ln(x)

```plot
Math.log(x)
```

## 组合函数

### cos(x) + sin(2x)

```plot
Math.cos(x) + Math.sin(2*x)
```

### e^(-x²) 高斯函数

```plot
Math.exp(-x*x)
```

### x · cos(x)

```plot
x * Math.cos(x)
```
