# 接口设计

> 转自 [mattpocock/skills](https://github.com/mattpocock/skills)，版权归原作者所有。

多方案并行接口设计流程。

## 流程

1. **识别问题** — 需要设计的接口
2. **生成方案** — 至少两个备选接口形状
3. **压力测试** — 用具体场景测试每个方案
4. **选择并记录** — 做出决策，写 ADR（如需要）

## 方案评估

对每个方案评估：

- **可测试性** — 能通过公共接口测试行为吗？
- **可扩展性** — 加新功能需要改接口吗？
- **可理解性** — 新用户能直觉地用对吗？
- **实现成本** — 当前方案背后实现有多复杂？
