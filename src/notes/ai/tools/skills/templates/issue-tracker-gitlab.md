# GitLab Issue 追踪

> 转自 [mattpocock/skills](https://github.com/mattpocock/skills)，版权归原作者所有。

GitLab 作为问题追踪器的使用约定。

## CLI 命令

- `glab issue list` — 列出 issue
- `glab issue view <number>` — 查看详情
- `glab issue create --title <t> --description <b>` — 创建
- `glab issue update <number> --label <l>` — 加标签
- `glab issue close <number>` — 关闭

## 约定

与 GitHub 类似，但使用 `glab` CLI。标签名映射可能不同——运行 `/setup-matt-pocock-skills` 确认。
