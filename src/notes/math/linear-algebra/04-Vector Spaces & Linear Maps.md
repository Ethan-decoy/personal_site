# 向量空间与线性映射

## 向量空间与子空间 (Vector Space & Subspace)

> 向量空间和子空间本质是相同的

### 零向量

- 过原点

### 加法封闭

- 假设 $u, v$ 在集合内，证明 $u+v$ 也满足集合条件

### 乘法封闭

- 假设 $u$ 在集合内，标量 $c$ 任意，证明 $cu$ 也满足集合条件

### 例：

**证明集合 $W = \{ (x_1, x_2, x_3, x_4) \in \mathbb{R}^4 \mid x_1 + x_2 - x_3 + 2x_4 = 0 \}$ 是 $\mathbb{R}^4$ 的子空间。**

**证明步骤：**

1. **验证包含零向量**  
    四维空间的零向量是 $\mathbf{0} = (0, 0, 0, 0)$。  
    将 $\mathbf{0}$ 的四个分量代入集合的定义方程中：  
    $0 + 0 - 0 + 2(0) = 0$  
  等式成立，因此 $\mathbf{0} \in W$。

2. **验证加法封闭**  
    假设 $u = (u_1, u_2, u_3, u_4)$ 和 $v = (v_1, v_2, v_3, v_4)$ 都是集合 $W$ 中的任意向量。  
    根据定义，已知它们满足方程：  
    ①$u_1 + u_2 - u_3 + 2u_4 = 0$  
    ②$v_1 + v_2 - v_3 + 2v_4 = 0$  
    考察它们的和向量：  
    $u + v = (u_1+v_1, u_2+v_2, u_3+v_3, u_4+v_4)$。  
    将 $u+v$ 的分量代入方程左边：  
    $(u_1+v_1) + (u_2+v_2) - (u_3+v_3) + 2(u_4+v_4)$  
    利用加法交换律重新分组，把 $u$ 和 $v$ 聚在一起：  
    $= (u_1 + u_2 - u_3 + 2u_4) + (v_1 + v_2 - v_3 + 2v_4)$  
    代入已知条件 ① 和 ②：  
    $= 0 + 0 = 0$  
    等式成立，因此 $u+v \in W$。

3. **验证数乘封闭**  
    假设 $u = (u_1, u_2, u_3, u_4)$ 是集合 $W$ 中的任意向量（即已知 $u_1 + u_2 - u_3 + 2u_4 = 0$）。  
    设 $c$ 为任意实数（标量）。  
    考察数乘向量：  
    $cu = (cu_1, cu_2, cu_3, cu_4)$。  
    将 $cu$ 的分量代入方程左边：  
    $(cu_1) + (cu_2) - (cu_3) + 2(cu_4)$  
    提取公因式$c$：  
    $= c(u_1 + u_2 - u_3 + 2u_4)$  
    代入已知条件：  
    $= c(0) = 0$  
    等式成立，因此 $cu \in W$。  

**结论**

因为 $W$ 满足包含零向量、加法封闭和数乘封闭这三个条件，所以 $W$ 是 $\mathbb{R}^4$ 的一个子空间。

### 子空间的几何形状

子空间必须是**平直**的（不能弯曲）：

| 维度 | $\mathbb{R}^2$ 的子空间 | $\mathbb{R}^3$ 的子空间 |
|------|------------------------|------------------------|
| 0维 | $\{原点\}$ | $\{原点\}$ |
| 1维 | 过原点的**直线** | 过原点的**直线** |
| 2维 | 整个 $\mathbb{R}^2$ | 过原点的**平面** |
| 3维 | — | 整个 $\mathbb{R}^3$ |

**注意**：曲线（如 $y=x^2$, $y=x^3$）即使过原点也不是子空间，因为加法不封闭。

## 线性无关与张成空间 (Linear Independence & Span)

### 线性组合 (Linear Combination)

把几个向量各自缩放（数乘），然后首尾相连（相加），得到一个新向量。

$$
c_1\vec{v}_1 + c_2\vec{v}_2 + \cdots + c_n\vec{v}_n
$$

### 张成空间 (Span)

所有可能的线性组合能到达的全部点。

**例子**：

- $\vec{v}_1 = (1, 0)$，$\vec{v}_2 = (0, 1)$ → $\text{span}\{\vec{v}_1, \vec{v}_2\} = \mathbb{R}^2$（整个平面）
- $\vec{v}_1 = (1, 0)$，$\vec{v}_2 = (2, 0)$ → $\text{span}\{\vec{v}_1, \vec{v}_2\} = x\text{轴}$（只有一条直线）

### 线性相关 (Linear Dependent)

至少有一个向量可以被其他向量线性组合表示（有冗余）。

**判断条件**：存在不全为零的系数 $c_1, c_2, \ldots, c_n$ 使得：

$$
c_1\vec{v}_1 + c_2\vec{v}_2 + \cdots + c_n\vec{v}_n = \vec{0}
$$

### 线性无关 (Linear Independent)

每个向量都贡献了一个新的、不可替代的方向。

**判断条件**：只有当所有 $c_i = 0$ 时，上式才成立。

**注意**：线性相关不仅是"共线"或"含零向量"，还包括更隐蔽的情况——比如 $\vec{v}_3 = \vec{v}_1 + \vec{v}_2$ 时，三个向量线性相关（即使没有两个共线）。

## 基与维数 (Basis & Dimension)

### 基 (Basis)

一组向量是某个向量空间的基，需要同时满足两个条件：

- 线性无关（没有冗余）
- 张成整个空间（够用，不缺）

**直觉**：基 = 描述这个空间的最小且够用的向量集合。

- 标准基 = $(1,0), (0,1)$ → **唯一**
- 任意基 = 任何满足"线性无关 + 张成"的向量组 → **无穷多**

但 $(2,1)$ 和 $(1,3)$ 也是 $\mathbb{R}^2$ 的一组基——同样无关且张成 $\mathbb{R}^2$。

**基不唯一，标准基唯一**

### 维数 (Dimension)

一个向量空间的维数 = 它任何一组基中向量的个数。

**关键定理**：所有基的大小都一样（向量个数相同）。

## 四个基本子空间 (The Four Fundamental Subspaces)

> 给定一个 $m \times n$ 矩阵 $A$，它天然关联着四个子空间。这是线性代数中最重要的结构化理解之一。

### 列空间 (Column Space)，记作 $C(A)$

= $A$ 的所有列向量的**张成空间**。

> 直观理解：**$A\vec{x}$ 能到达的所有 $\vec{b}$ 的集合**。

- 在 $\mathbb{R}^m$ 中
- 维数 = 矩阵的秩 (rank)
- **用途**：判断 $A\vec{x} = \vec{b}$ **有无解**（$\vec{b}$ 必须在列空间里）

### 行空间 (Row Space)，记作 $C(A^T)$

= $A$ 的所有行向量的张成空间 = $A^T$ 的列空间。

> 直观理解：从"行"的角度看矩阵能张成什么。

- 在 $\mathbb{R}^n$ 中
- 维数 = rank（和列空间维数一样）
- **用途**：判断哪些约束是独立的（独立方程个数）

### 零空间 (Null Space)，记作 $N(A)$

= 所有满足 $A\vec{x} = \vec{0}$ 的解 $\vec{x}$ 组成的集合。

> 直观理解：**哪些输入被矩阵"压缩没了"（变成零向量）**。

- 在 $\mathbb{R}^n$ 中
- 维数 = $n$ - rank（叫零化度 nullity）
- **用途**：判断**解的个数**
  - 零空间 = $\{\vec{0}\}$ → 解唯一
  - 零空间有非零向量 → 无穷多解

### 左零空间 (Left Null Space)，记作 $N(A^T)$

= 所有满足 $A^T\vec{y} = \vec{0}$ 的解 $\vec{y}$ 组成的集合。

等价于 $\vec{y}^T A = \vec{0}^T$，所以叫"左"零空间（$\vec{y}^T$ 在 $A$ 的**左边**）。

> 直观理解：哪些向量与 $A$ 的**每一列都正交**。

- 在 $\mathbb{R}^m$ 中
- 维数 = $m$ - rank
- **用途**：判断 $\vec{b}$ 要满足什么**相容性条件**方程才有解

### 核心定理：Rank-Nullity Theorem

$$
\text{rank}(A) + \text{nullity}(A) = n
$$

即：列空间维数 + 零空间维数 = 列数 $n$

### 四个子空间的作用总结

| 子空间 | 回答的问题 | 实际用途 |
|--------|-----------|---------|
| **列空间** $C(A)$ | $\vec{b}$ 在不在 $A$ 能到达的范围里？ | 判断**有无解** |
| **零空间** $N(A)$ | 解唯一吗？还是有无穷多解？ | 判断**解的个数** |
| **行空间** $C(A^T)$ | 哪些约束是独立的？ | 判断**独立方程个数** |
| **左零空间** $N(A^T)$ | $\vec{b}$ 要满足什么条件？ | 判断**相容性条件** |

### 例子

$A = \begin{bmatrix} 1 & 2 \\ 2 & 4 \end{bmatrix}$（$2 \times 2$ 矩阵）

- 第二列 = 2 × 第一列 → 线性相关 → **rank = 1**
- **列空间**：第一列 $(1,2)$ 张成的直线（在 $\mathbb{R}^2$ 中）
- **行空间**：第一行 $(1,2)$ 张成的直线（在 $\mathbb{R}^2$ 中）
- **零空间**：$x_1 + 2x_2 = 0 \implies x_1 = -2x_2$，也是一条直线（在 $\mathbb{R}^2$ 中）
- **左零空间**：与列空间正交的方向

维数验证：rank + nullity = 1 + 1 = 2 = n ✓

### 零空间与解的关系

如果 $\vec{x}_0$ 是 $A\vec{x} = \vec{b}$ 的一个解，零空间里有向量 $\vec{n}$，那么：

$$
A(\vec{x}_0 + \vec{n}) = A\vec{x}_0 + A\vec{n} = \vec{b} + \vec{0} = \vec{b}
$$

所以 $\vec{x}_0 + \vec{n}$ 也是解。

**通解公式**：$\vec{x} = \vec{x}_0 + t \cdot \vec{n}$（$t$ 任意实数，$\vec{n}$ 为零空间基向量）
