# Wiki

## IDL

IDL（Interface Definition Language）是一种**简化的描述语言**，用于描述软件组件之间的接口格式，是一种独立于具体编程语言的“数据契约”或“通用图纸”。

是为了解决跨进程/跨语言（如 C++ 和 Python）通信时，双方内存结构不互通的问题。

开发者只写一次 IDL 描述数据，工具自动生成各语言的“打包/解包”代码。

> 一次定义，多端互通。

## C/S架构

Client / Server 架构。

一种中心化的分布式设计。客户端负责交互与请求，服务端负责集中处理与资源管理。

流程：Client 发起请求 → 网络传输 → Server 处理业务 → 返回结果

> 一问一答，强依赖。服务端是”大脑”，客户端是”手脚”；服务端挂了，所有客户端瘫痪。

## 夏琨塔拉·戴维

夏琨塔拉·戴维（Shakuntala Devi，1929–2013），印度心算家，被誉为”人类计算机”。

她能在几秒内完成两个 13 位随机数的乘法、心算 201 次方开 23 次方，速度超过当时的电子计算机。吉尼斯世界纪录收录其多项心算纪录。

> 人类大脑的算力天花板。

## 平信
无连接、不保证送达、没有回执（发出去就不管了，丢了不补）。

省去了确认和重传的机制开销，速度极快，延迟极低。
